3. A weboldalam egy hajóutakat szervező fiktív társaság számára készült, akiknek a fő profilja, hogy évente megújuló téli és nyári hajóutakat szerveznek Európában. A weboldalamon megtalálható egy általános leírás a cég tevékenységéről, valamint az aktuális téli és az aktuális nyári utazás. Ezen felül van egy kapcsolat felvételre alkalmas űrlap, illetve egy árkalkulátor.
4. A saját Java Script programom az arak.html oldalon található. A program funkciója, hogy a felhasználó által kiválasztott túratípus, szobatípus, egyes szolgáltatástípusok és egy vagy  több kiválasztott kedvezménytípus alapján kiszámolja az utazás egy főre várható összegét. Ezen felül megmutatja, hogy ebből az egyes tételek mekkora összeget tesznek ki. Továbbá kiírja, hogy az 1, 2 vagy 3 különböző kedvezmény felhasználása mekkora megtakarítást jelent. Látványelemként a túra típusától függő kép jelenik meg az oldalon.
5. A teljes weboldalon abhaya-libre betűtípust használtam. A cím az extrabold, a menüpontok a semibold, a tartalom pedig a regular változata.
